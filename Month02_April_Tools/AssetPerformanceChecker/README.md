# 资产性能检查工具 (Asset Performance Checker)

## 功能
自动检查 Maya 场景中所有模型的：
- 面数 / 顶点数统计
- UV 是否在 0-1 象限
- 非流形几何检测
- 变换冻结状态
- 资产间距分析

## 输出
生成深色主题 HTML 报告，绿/黄/红三色标记，每项附中文修复建议。

## 运行方式
1. Maya 脚本编辑器运行 `asset_perf_check.py`
2. 无选中 → 弹窗选"一键全场景扫描"
3. 已选中 → 只检查选中物体
4. 自动打开浏览器显示报告
